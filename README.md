# Student Management System
A JavaFX + JDBC + PostgreSQL CRUD application.

## Features
- Login screen (Scene 1)
- Student Record Management (Scene 2)
- Add, Update, Delete, View students
- Connected to PostgreSQL database
- Environment file (config.properties) for credentials

## Tech Stack
- Java 17
- JavaFX 21
- PostgreSQL 18
- JDBC
- Maven
